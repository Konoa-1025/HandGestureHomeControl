# HandGestureHomeControl with ECHONET Lite


## 概要
ハンドジェスチャーでECHONET Lite対応家電を操作するシステムです。

## 特徴
- MediaPipeによる手認識
- 複数のネットワークカメラ対応
- TCP通信によりログは遠くでも取得可能
- ECHONET Lite制御
- CPU負荷によるモデルの切り替え
- メモリ使用量によるカスケード切り替え
- ターゲットコンボによる家電の指定

## 動作条件
- Python 3.9以上
- OpenCV
- MediaPipe
- psutil
- json
- art
