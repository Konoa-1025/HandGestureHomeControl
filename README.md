# HandGestureHomeControl Python3.11
**No remote. No voice. Just your hand.**

「あれ。リモコンどこ？」

この一言を無くし

「俺の力をみよ！電気を消すぞ！」

という楽しんでいる言葉を増やす

このようなコンセプトで作られているプロジェクト

## ✨Overview
HandGestureHomeControlは、ハンドジェスチャーだけで家電を操作するスマートホームシステムです！

あの映画の魔法使いのように。

ナルトの印を結ぶように。

**楽しんで手を動かすだけで家電を操作できます！**

リモコンを探す時間も、スマホ取り出す必要も、重い腰をあげて立つ必要すらありません！！

## 🧐Why?
こんな経験ない？
- 料理で手が油まみれ
- 「うっわボタンきったね」
- 寒いからエアコンつけたいけど布団から出れない
- リモコンは何処へ・・・

**こんな時に、手をかざすだけ。**

これがこのプロジェクトの目標です。

## How does it work?
```
指を刺して操作したい家電を指定
⇩
グー・チョキ・パー　この三種類のコマンドを覚えればいいのです。
⇩
操作成功
```
**かなりシンプル！**

印を結ぶより簡単です！

## Features
- リアルタイムハンドジェスチャー認識
- 指差しによる家電選択
- ターゲットコンボ方式
- ECHONET Lite対応
- CPU負荷によるモデルの自動切り替え
- ネットワークカメラ対応
