# HandGestureHomeControl

## 概要
ハンドジェスチャーだけで家電を操作するシステムです！
- 非接触型で魔法やナルトの印結びのように操作が可能で、その場から動かなくても家電を操作できるのが魅力になります！
- 家事をしてて手が汚れていたり、感染症が不安な場所であったり、寝たいけどライト消すのにリモコンを探さなくちゃいけなくて辛い人におすすめです！

## 特徴
- MediaPipeによる手認識をしています
- 複数のネットワークカメラ対応
- TCP通信によりログは遠くでも取得可能
- ECHONET Lite制御
- CPU負荷によるモデルの切り替え
- メモリ使用量によるカスケード切り替え
- ターゲットコンボによる家電の指定

## 動作条件
- Python 3.9以上
- OpenCV
- MediaPipe
- psutil
- json
- art
